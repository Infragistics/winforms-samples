Project Manager for Windows Forms

Infragistics コントロールを使用するプロジェクトを作成しました。

注目のコントロール:

  • WinGanttView

  • WinGanttViewPrintDocument

  • WinToolbarsManager

  • WinTouchProvider

このアプリケーションは以下の機能を使用します:

  • UltraToolbarsManager を使用する Office 2013 スタイルのリボン
  • DataSet にバインドされる UltraGanttView およびリボンのツールによってプロジェクト変更操作
  • 複数の定義済みテーマ
  • タッチ対応
  • UltraGanttView の印刷

問題 推奨事項
Infragistics WinForms 製品はシステムにインストールされていません。 Infragistics WinForms 製品をインストールしてください。jp.infragistics.com から無料トライアル版をダウンロードできます。
Infragistics アセンブリ (DLL) 参照を指定したバージョンにアップデートする必要があります。

デフォルトで、テンプレートは最新の Infragistics WinForms ボリューム リリースを使用します。別の互換性のあるバージョンを使用するには、アセンブリをアップデートする必要があります。このテンプレートの最小の互換性のあるバージョンは 14.1 です。

また、アップデートするためにバージョン ユーティリティを使用できます。
