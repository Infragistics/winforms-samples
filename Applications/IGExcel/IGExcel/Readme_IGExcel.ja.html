IG Excel for Windows Forms

エクスポートが完了しました。Infragistics コントロールを使用するプロジェクトを作成しました。

注目のコントロール:

  • WinSpreadsheet

  • WinToolbarsManager

このアプリケーションは以下の機能を使用します:

  • WinSpreadsheet コントロールは Excel のレイアウトおよび動作をエミュレートするために使用されます。
  • WinToolbarsManager コンポーネントは Office 2013 のリボン UI を作成するために使用されます。

問題 推奨事項
Infragistics WinForms 製品がシステムにインストールされていません。 Infragistics WinForms 製品をインストールしてください。jp.infragistics.com から無料トライアル版をダウンロードできます。
Infragistics アセンブリ (DLL) 参照を指定したバージョンにアップデートする必要があります。

デフォルトで、テンプレートは最新の Infragistics WinForms ボリューム リリースを使用します。別の互換性のあるバージョンを使用するには、アセンブリをアップデートする必要があります。このテンプレートの最小の互換性のあるバージョンは 16.1 です。

また、このプロセスを自動化するためにバージョン ユーティリティを使用できます。
