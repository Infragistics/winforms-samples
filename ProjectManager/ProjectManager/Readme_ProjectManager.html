Troubleshooting:

Issue Recommendations
The Infragistics WinFormd product is not installed on your system. Verify the Infragistics WinForms product is installed. To download a free trial, please visit www.infragistics.com.
The Infragistics assembly (DLL) references must be updated to the correct/desired version.

By default, the template uses the latest major/volume release of Infragistics WinForms. If you want to use a different compatible version, you must update the assemblies. The minimum compatible version for this template is 14.1.

Or you can use the version utility to automate this process.
